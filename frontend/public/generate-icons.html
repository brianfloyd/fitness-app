Icon Generator for Fitness Daily Log

This tool generates PNG icons from the SVG favicon for iOS home screen and PWA support.

iOS apple-touch-icon

Android/PWA

PWA Large

Instructions:

  1. Click each "Download" button to save the PNG icon
  2. Save the files in the frontend/public/ directory with these names:
    • icon-180x180.png
    • icon-192x192.png
    • icon-512x512.png
  3. Refresh your app - the icons should now work for iOS home screen!
